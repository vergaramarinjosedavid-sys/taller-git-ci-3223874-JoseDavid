# Taller Git & CI - SENA

Proyecto del Taller Integrador (Semanas 1 a 6).

## Contenido

- Estructura web básica (`index.html`)
- Función de suma + pruebas unitarias con `node:test`
- Prettier configurado para formato automático
- Workflow CI con Prettier + tests

## Scripts disponibles

- `npm test`
- `npm run prettier:check`
- `npm run prettier:write`
